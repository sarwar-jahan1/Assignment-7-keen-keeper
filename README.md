# 🌱 KeenKeeper

## 📌 Project Description

KeenKeeper is a personal relationship management web application designed to help users maintain meaningful connections. It allows users to track interactions with friends, monitor relationship status, and stay consistent in communication.

---

## 🚀 Technologies Used

* **React.js** – for building the user interface
* **React Router** – for navigation and routing
* **Tailwind CSS** – for styling and responsive design
* **React Icons** – for UI icons
* **Recharts** – for data visualization (charts)
* **Context API** – for global state management

---

## ✨ Key Features

### 1️⃣ Relationship Tracking

Track how long it has been since you last interacted with a friend and identify who needs attention.

### 2️⃣ Interaction Logging (Timeline)

Log interactions such as Call, Text, and Video. Each action is stored and displayed in a timeline view.

### 3️⃣ Analytics Dashboard

Visualize your communication habits through dynamic charts showing interaction types.

---

## 📷 UI Highlights

* Clean and responsive design
* Interactive cards for each friend
* Real-time toast notifications on actions
